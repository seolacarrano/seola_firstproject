YUMYUM
This is a Yelp clone built by React, JavaScript, and Yelp API. Users can search for restaurants in any place that Yelp offers.
Error Note
This is a wrong answer note. Users sign up and login with a custom username and password and then they can add a wrong answer note. Each user can create, read, update, and delete their note. This project was built using Python, Django, Vue CLI, and Buefy.
Tello
This project is a clone of the popular productivity app Trello. Users sign up and login with a custom username and password. Each user utilizes CRUD (i.e. create, read, update, and delete) functionality to manipulate each card and/or board on a backend server created and maintained by Ruby on Rails and SQL.
My Album
This is an album that users can add, edit, and delete images and memo. This project is built using CSS, HTML and JavaScript jQuery and responsive to different devices such as mobile, tablet, and desktop.
